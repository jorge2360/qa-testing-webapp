# QA Testing WebApp

Proyecto orientado a pruebas QA manuales y validación funcional de aplicaciones web.

---

## Objetivo

Demostrar conocimientos básicos de:

- testing manual
- pruebas funcionales
- validación frontend/backend
- testing API
- documentación QA
- reportes de bugs
- pruebas CRUD
- validaciones de autenticación

---

## Sistema bajo prueba

TaskFlow Fullstack

Tecnologías evaluadas:

- Laravel API
- React
- PostgreSQL
- Sanctum Authentication

---

## Áreas evaluadas

- Login
- Registro
- CRUD de tareas
- Validaciones
- Protección de rutas
- Persistencia de sesión
- API Testing
- Manejo de errores
